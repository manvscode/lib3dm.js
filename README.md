# m3d.js
This is a JavaScript port of m3d for WebGL applications.

## Supported Constructs
* 2D Vectors
* 3D Vectors
* 4D Vectors
* 2x2 Matrices
* 3x3 Matrices
* 4x4 Matrices
* Quaternions
* Transformations
* Geometric Tools

##  Build Instructions

To build, run this command:

   npm run build

### Unix-like OSs
Just run ./build.sh
